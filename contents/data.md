# 데이터모델링
- 데이터모델은 마이크로서비스 내부구조가 도메인 모델이 아닌 경우는 독자적으로 진행하며, 도메인모델 기반인 경우 도메인 모델을 고려하여 설계해야 한다. 
- 이 때 과제는 도메인모델을 어떻게 저장소에 효율적으로 매핑할 것이냐는 것이다.
- 도메인 모델,JPA 인 경우 OR매핑 가이드를 참고한다.
