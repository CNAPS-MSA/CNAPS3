# DevOps
- Devops란?
  - 최종 사용자에게 가치를 지속적으로 제공할 수 있는 사람, 프로세스 및 제품의 결합체. 

- Devops Infra 또는 Devops tool 란?
  - 사용자에게 가치를 지속적으로 제공하는 작업을 수행하는 데 필요한 도구를 제공하는 서비스 세트.
  - 애플리케이션을 클라우드 또는 온-프레미스에 빌드, 테스트 및 배포 가능. 
  - 투명성, 협업, 지속적인 업데이트 및 지속적인 배포를 지원
